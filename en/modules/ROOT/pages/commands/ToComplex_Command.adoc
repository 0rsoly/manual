= ToComplex Command

ToComplex( <Vector> )::
  Transforms a vector or point to a complex number in algebraic form.
  +

[EXAMPLE]

====

`ToComplex((3, 2))` yields _3 + 2ί_.

====

== [#CAS_Syntax]#CAS Syntax#

ToComplex( <Vector> )::
  Transforms a vector or point to a complex number in algebraic form.
  +

[EXAMPLE]

====

`ToComplex((3, 2))` yields _3 + 2ί_.

====

[NOTE]

====

* The complex ί is obtained by pressing [.kcode]#ALT # + [.kcode]# i#.
