= Dimension Command

Dimension( <Object> )::
  Gives the dimension of a vector or a Matrix.
  +

[EXAMPLE]

====

*Example:*

`Dimension({1, 2, 0, -4, 3})` yields _5_.

====

[EXAMPLE]

====

*Example:*

`Dimension({{1, 2}, {3, 4}, {5, 6}})` yields _\{3, 2}_.

====
