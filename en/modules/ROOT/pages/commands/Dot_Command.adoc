= Dot Command

Dot( <Vector>, <Vector> )::
  Returns the dot product (scalar product) of the two vectors.
  +

[EXAMPLE]

====

*Example:*

`Dot({1, 3, 2}, {0, 3, -2})` yields _5_, the scalar product of _\{1, 3, 2}_ and _\{0, 3, -2}_.

====

[NOTE]

====

*Note:*

See also xref:/commands/Cross_Command.adoc[Cross Command].

====
