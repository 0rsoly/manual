= FitLine Command

FitLine( <List of Points> )::
  Calculates the _y_ on _x_ regression line of the points.
  +

[EXAMPLE]

====

*Example:* `FitLine({(-2, 1), (1, 2), (2, 4), (4, 3), (5, 4)})` yields _0.4x + 2_.

====
