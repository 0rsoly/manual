= Eigenvectors Command

== [#CAS_Syntax]#CAS Syntax#

Eigenvectors( <Matrix> )::
  Finds the eigenvectors of the given matrix.

[EXAMPLE]
====

`Eigenvectors({{1, 2}, {3, 4}})` yields stem:[ \left(\begin\{array}\sqrt\{33} - 3&-\sqrt\{33} -
3\\6&6\\\end\{array}\right) ]

====
