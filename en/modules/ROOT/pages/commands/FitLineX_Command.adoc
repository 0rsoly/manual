= FitLineX Command

FitLineX( <List of Points> )::
  Calculates the _x_ on _y_ regression line of the points.

[EXAMPLE]
====

`++FitLineX({(-1, 3), (2, 1), (3, 4), (5, 3), (6, 5)})++` yields _1.1x - 0.1_.

====
