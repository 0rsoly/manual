= NextPrime Command
:page-en: commands/NextPrime

NextPrime( <Number> )::
  Returns the smallest prime greater than the entered number.

[EXAMPLE]
====

`++NextPrime(10000)++` yields _10007_.

====

[NOTE]
====

See also xref:/commands/PreviousPrime.adoc[PreviousPrime Command].

====
