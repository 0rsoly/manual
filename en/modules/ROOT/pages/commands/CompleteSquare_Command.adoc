= CompleteSquare Command

CompleteSquare( <Quadratic Function> )::
  Returns the quadratic function in the form: stem:[a (x - h)^2 + k].

[EXAMPLE]
====

`CompleteSquare(x^2 - 4x + 7)` yields _1 (x - 2)^2^ + 3_.

====
