= SetTrace Command
:page-en: commands/SetTrace
ifdef::env-github[:imagesdir: /en/modules/ROOT/assets/images]

SetTrace( <Object>, <true | false> )::
  Turns xref:/Tracing.adoc[Tracing] on/off for the specified object.

[EXAMPLE]
====

Create a point A, then type in `++SetTrace(A,true)++`. Select the image:22px-Mode_move.svg.png[Mode
move.svg,width=22,height=22]xref:/tools/Move.adoc[Move Tool] and drag the point, to show its trace.

====

[NOTE]
====

Use xref:/commands/ZoomIn.adoc[ZoomIn(1)] to clear all traces.

====
