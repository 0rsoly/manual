= SetSeed Command

SetSeed( <Integer> )::
  Seeds the random number generator so that subsequent random numbers will be determined by the seed.

[EXAMPLE]
====

`SetSeed(33)`

====

====

command], xref:/commands/RandomUniform_Command.adoc[RandomUniform command].
