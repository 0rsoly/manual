= Slope Tool

image:16px-Menu_view_graphics.svg.png[Menu view graphics.svg,width=16,height=16] xref:/Graphics_View.adoc[Graphics
View], whose size may be changed using xref:/Properties_Dialog.adoc[Properties Dialog]
