= Comando Dimensione

Dimensione(Oggetto)::
  Determina la dimensione di un vettore o di una matrice.
  +

[EXAMPLE]

====

*Esempi:*

`Dimensione({1, 2, 0, -4, 3})` restituisce 5.

`Dimensione({{a, b}, {c, d}, {e, f}})` restituisce \{3, 2}.

====
