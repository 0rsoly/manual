= Comando ListaDivisori

ListaDivisori(Numero)::
  Restituisce la lista di tutti i divisori positivi del numero, incluso il numero stesso.

[EXAMPLE]
====

`++ListaDivisori(15)++` restituisce \{1, 3, 5, 15}.

====
