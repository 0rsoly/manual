= Comando RegPot

RegPot(Lista di punti)::
  Genera la curva di regressione nella forma _a x^b^_.

[EXAMPLE]
====

`++RegPot({(1, 1), (3, 2), (7, 4)})++` genera la curva di regressione _f(x) = 0.97 x^0.71^_.

====

== Sintassi CAS

RegPot(Lista di punti)::
