= Comando Autovettori

== [#Sintassi_CAS]#Sintassi CAS#

Autovettori( Matrice )::
  Determina gli autovettori della matrice indicata.
  +

[EXAMPLE]

====

*Esempio:* `Autovettori({{1, 2}, {3, 4}})` restituisce stem:[ \left(\begin\{array}\sqrt\{33} - 3&-\sqrt\{33} -
3\\6&6\\\end\{array}\right) ]
