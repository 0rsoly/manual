= Comando Sfera

Sfera(Punto, Raggio)::
  Genera la sfera avente centro e raggio indicati.

Sfera(Punto, Punto)::
  Genera la sfera avente centro nel primo punto e passante per il secondo punto indicato.

[EXAMPLE]
====

`Sfera((0, 0, 0), (1, 1, 1))` restituisce _x² + y² + z² = 3_
