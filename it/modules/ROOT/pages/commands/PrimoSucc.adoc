:page-en: commands/NextPrime

PrimoSucc(Numero)::
  Restituisce il minore tra i numeri primi maggiori del numero dato.

[EXAMPLE]
====

`++PrimoSucc(10000)++` restituisce _10007_.

====

[NOTE]
====

Vedere anche il comando xref:/commands/PrimoPrec.adoc[PrimoPrec].

====
