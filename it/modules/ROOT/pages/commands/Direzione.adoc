= Comando Direzione

Direzione(Retta)::
  Determina il vettore direzione della retta.
  +

[EXAMPLE]
====

`++Direzione(-2x + 3y + 1 = 0)++` restituisce il vettore stem:[u= \begin\{pmatrix} 3 \\ 2 \end\{pmatrix} ]

====

[NOTE]
====

Una retta di equazione _ax + by = c_ ha il vettore direzione di componenti _(b, - a)_.

====
