= Comando CopiaOggettoLibero

CopiaOggettoLibero(Oggetto)::
  Genera una copia xref:/Oggetti_liberi,_dipendenti_e_ausiliari.adoc[libera] dell'oggetto indicato. Gli oggetti copiati
  mantengono tutte le xref:/Propriet%C3%A0_degli_oggetti.adoc[proprietà degli oggetti] di base. La copia di un
  xref:/Oggetti_liberi,_dipendenti_e_ausiliari.adoc[oggetto ausiliare] è un oggetto ausiliare.
