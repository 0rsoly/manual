= Comando BaseInf

BaseInf(Quadrica)::
  Genera la base inferiore della quadrica indicata.

[EXAMPLE]
====

`++BaseInf(cilindro)++` genera una circonferenza.

====

[NOTE]
====

Vedere anche i comandi xref:/commands/BaseSup.adoc[BaseSup], xref:/commands/SuperficieLaterale.adoc[SuperficieLaterale]
e xref:/commands/BasiInfSup.adoc[BasiInfSup].

====
