SigmaYY(Lista di punti)::
  Calcola la somma dei quadrati delle ordinate dei punti indicati.

[EXAMPLE]
====

Sia `++lista1 = { (-3, 4), (-1, 4), (-2, 3), (1, 3), (2, 2), (1, 5) }++` una lista di punti. `++SigmaYY(lista1)++`
restituisce _a = 79_.
