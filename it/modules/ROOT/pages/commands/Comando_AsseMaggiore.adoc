= Comando AsseMaggiore

AsseMaggiore(Conica)::
  Determina l'equazione dell'asse maggiore della conica.
  +
  +

[EXAMPLE]

====

*Esempio:* `AsseMaggiore(x^2 / 9 + y^2 / 4 = 1)` restituisce _y_ = 0.

====

[NOTE]

====

*Note:* Vedere anche il comando link:/it/Comando_AsseMinore[AsseMinore].

====
