BaseSup(Quadrica)::
  Genera la base superiore di una quadrica limitata.

[EXAMPLE]
====

* `++BaseSup(cilindro)++` genera una circonferenza.
* `++BaseSup(cono)++` genera un punto (il vertice del cono).

====

[NOTE]
====

Vedere anche i comandi xref:/commands/BaseInf.adoc[BaseInf], xref:/commands/SuperficieLaterale.adoc[SuperficieLaterale]
e xref:/commands/BasiInfSup.adoc[BasiInfSup].

====
