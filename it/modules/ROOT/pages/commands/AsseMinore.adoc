AsseMinore(Conica)::
  Determina l'equazione dell'asse minore della conica.

[EXAMPLE]
====

`++AsseMinore(x^2 / 9 + y^2 / 4 = 1)++` restituisce _x_ = 0

====

[NOTE]
====

Vedere anche il comando xref:/commands/AsseMaggiore.adoc[AsseMaggiore].

====
