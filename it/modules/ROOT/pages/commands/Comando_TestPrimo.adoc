= Comando TestPrimo

TestPrimo(Numero)::
  Restituisce _true_ se il numero è primo, _false_ in caso contrario.

[EXAMPLE]

====

* `TestPrimo(10)` restituisce _false_
* `TestPrimo(11)` restituisce _true_.

====
