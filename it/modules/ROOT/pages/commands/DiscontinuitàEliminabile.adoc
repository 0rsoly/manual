DiscontinuitàEliminabile( Funzione )

Determina e visualizza il punto in cui una funzione razionale presenta una discontinuità eliminabile.

[EXAMPLE]
====

`++DiscontinuitàEliminabile((3-x)/(2 x^(2)-6 x))++` restituisce _(3,-0.17)_.

====
