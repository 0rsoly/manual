= Comando Rimuovi

Rimuovi(Lista, Lista)::
  Rimuove dalla prima lista gli oggetti elencati nella seconda lista.

[EXAMPLE]
====

`++Rimuovi({1, 3, 4, 4, 9}, {1, 4, 5})++` restituisce la lista `++{3, 4, 9}++`.

====
