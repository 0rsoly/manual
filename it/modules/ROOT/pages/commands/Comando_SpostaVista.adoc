= Comando SpostaVista

SpostaVista(x, y)::
  Sposta la vista corrente di _x_ pixel a destra e di _y_ pixel verso l'alto.
SpostaVista(x, y, z)::
  Sposta la vista corrente di (_x_, _y_, _z_) nella _vista 3D_, oppure di (_x_, _y_) nella _vista Grafici_
  bidimensionale.
