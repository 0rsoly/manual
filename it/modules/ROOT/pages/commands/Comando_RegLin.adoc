= Comando RegLin

RegLin(Lista di punti)::
  Genera la retta di regressione di _y_ rispetto a _x_ dei punti indicati.
  +

[EXAMPLE]

====

`RegLin({(-2, 1), (1, 2), (2, 4), (4, 3), (5, 4)})` genera la retta _0.4x + 2_.
