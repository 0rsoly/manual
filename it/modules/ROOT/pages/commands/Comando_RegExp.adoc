= Comando RegExp

RegExp(Lista di punti)::
  Genera la curva di regressione esponenziale nella forma stem:[ a e ^ \{ b x } ]

[EXAMPLE]
====

`RegExp({(0, 1), (2, 4)})` restituisce stem:[e^\{0.69 x}].

====

[NOTE]
====
