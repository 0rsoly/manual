Baricentro(Poligono)::
  Calcola il baricentro del poligono.

[EXAMPLE]
====

Siano `++A = (1, 4)++`, `++B = (1, 1)++`, `++C = (5, 1)++` e`++D = (5, 4)++` i vertici del poligono
`++poli1=Poligono(A,B,C,D)++`. `++Baricentro(poli1)++` genera il punto _E = (3, 2.5)_.
